# 旅行雷达 · 一个人的旅行参谋部

为独行旅行者做的私人旅行决策与规划 PWA。零广告、零佣金、零上报——推荐只对你的口味负责。

## 它做三件事

1. **决策**（雷达）：出发地 × 月份 × 天数 × 16 种偏好 → 从 217 个目的地收敛成 3 个答案，每个都给大白话理由（"10月正当季""高铁 2 小时""契合你选的美食、秋色"）和诚实预警（"2 天玩不透""节假日人多"）
2. **深读**（档案）：90 城 × 14 区块手工编研档案——必吃、老字号、本地人心水、老街小巷、物候历、野生小友、住宿片区怎么选、防坑清单、入乡随俗、本地人玩法
3. **落地**（行囊）：行程草稿（慢游/标准/特种兵三档节奏，逐条高德导航直达）、分项预算、气候感知装备清单、独行安全卡（截图发家人）

## 特点

- **纯静态零依赖**：双击 `index.html` 就能用（file://），线上是 GitHub Pages
- **PWA**：手机浏览器「添加到主屏幕」即获得类小程序体验，离线可用
- **数据私有**：收藏、行程、笔记全存本机 localStorage，支持 JSON 导出/导入
- **纸墨雷达设计**：纸墨编辑风 + 暗金夜色双主题，系统宋体零网络字体

## 开发

```
data 管线：python3 _dev/tools/build_data.py   （_dev/data-src/*.json → data/*.js）
引擎自检：node _dev/selftest.js               （38 断言）
双端走查：python3 -m http.server 8123 & node _dev/verify.js（Playwright 截图+交互断言）
图标生成：python3 _dev/tools/make_icons.py
```

无构建、无框架：`index.html` + 6 个 JS（util/engine/fx/views/app + data 两片）。`data/deep.js`(453KB) 首屏后异步注入，file:// 与 https 双兼容。

## 数据说明

城市档案为 2024-2026 手工编研，非实时数据；营业时间、交通班次出行前请以官方为准。坐标精度 ±0.3°，仅用于距离估算与星座图。
