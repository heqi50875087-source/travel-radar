# 旅行雷达 · 2026 重建蓝图

> 2026-07-10 · 按斯坦福 STORM 方法四视角调研后定案。推翻旧版全部界面与架构，只回收数据资产。

## 一、定位

**一个人的旅行参谋部。** 不卖票、不种草、不推流——只做三件事：
1. **决策**：这个月去哪？（雷达推荐引擎，大白话理由）
2. **深读**：去了怎么玩透？（90 城 14 区块深度档案：必吃/防坑/本地人玩法/物候/老字号）
3. **落地**：行程草稿 + 预算 + 装备清单，全部离线可用、数据私有

对比商业产品的三个「反着来」：无广告无种草噪音（内容即结论）、离线优先数据不出本机（localStorage）、为一个人设计（一人食、独行安全、自由节奏）。

## 三、信息架构（手机底部 4 tab = 桌面顶栏 4 区）

```
#/radar   雷达 —— 出发地/月份/天数/预算档/16偏好胶囊 → 排名推荐卡(大白话理由+预警) → 国内/国际严格分栏
```

## 四、推荐引擎（engine.js 已实现）

评分 = 50 基线 + 季节(±22) + 距离(球面距离分档，长假远途豁免，周末远途重罚) + 偏好命中(16 胶囊×关键词词典，≤20) + 天数适配(±8) + 预算档(±6) + 深度档案加成(+5) + 免签加成(+6)。
输出**大白话理由**（"11月正当季""高铁2小时""契合你选的美食、古镇"）+ **诚实预警**（"人均偏贵""节假日人多""2天玩不透"）。绝不显示裸分数。

## 五、工程架构

- 纯静态零构建：plain `<script>` 全局命名空间 `TR`，file:// 双击可用 + GitHub Pages 线上
- 数据两段加载：core.js 同步（首屏即用）→ deep.js 异步（档案页就绪制，`TR.whenDeep`）
- PWA：manifest + sw.js（stale-while-revalidate），添加到主屏幕 = 类小程序
- localStorage 命名空间 `tr2026.*`：favs / trips / notes / settings
- 特效遵守 prefers-reduced-motion，canvas 用 IntersectionObserver 惰性启停
- 部署：GitHub Pages（heqi50875087-source），Chinese 文件名不入 URL（全 ASCII 文件名）

## 六、设计系统 ·「纸墨雷达」（STORM 设计视角定案）

**方向**：纸墨编辑风为骨 + 2026 编辑排版语言（衬线复兴/极端层级对比/bento 网格/tactile 纸感），唯一签名交互 = Canvas 雷达罗盘。暗色 = 环游志暗金反相。

**浅色**：纸 #F6F1E9 / 卡面 #FCFAF5 / 墨 #211C16 / 墨2 #5B5147 / 赤陶 #C75C3D（主行动·扫描线）/ 松绿 #2F4A3F / 黄铜 #B08A4F（发丝线）/ 天青 #8AA6A8
**暗色**：底 #17130E / 面 #221C15 / 字 #ECE3D4 / 次 #A99A85 / 赤陶提亮 #E0714E / 金 #D4AF63（暗色主角）/ 松绿 #6E8F7E

**字体**（零网络字体）：标题正文衬线 Songti SC/Source Han Serif；UI 标签 system-ui/PingFang SC。城市名 clamp(2.75rem,8vw,6rem)，元数据 0.8125rem 字距 +0.08em——刻意极差。

**质感**：4px 基网；卡 14px/胶囊 999px；1px 黄铜发丝描边 + 极淡阴影；玻璃只用于顶栏/底 tab/抽屉；纸纹 + 等高线暗纹。

**动效**：微交互 120-160ms/卡片 240-320ms/进场 400-600ms 错峰；缓动 cubic-bezier(.22,1,.36,1)；reduce-motion 退静态。

**特效白名单**：雷达罗盘、城市星座图、滚动揭示、桌面卡片微倾斜、hero 轻视差。**黑名单**：3D 地球、通篇玻璃、靛紫渐变、自动轮播、emoji 堆 UI。

**移动**：底 4 tab 玻璃栏（雷达/城市/行囊/我的）+ 档案全屏 sheet + 100dvh + safe-area。**桌面**：顶栏 + 杂志两栏 + hover，max-width 1240。

## 七、明确不做

- 登录/账号/云同步（localStorage + 导出足矣）
- 实时票价酒店（商业 API 领域，链接跳转即可）
- 评论/社区（一个人的工具）
- AI 在线生成（保持零依赖零费用；数据即内容）
